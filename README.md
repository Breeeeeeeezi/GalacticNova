# Galactic Nova

A curated collection of browser games, made with a purple theme, floating particles, overlay player, favorites, and background music.  

## How to use:
- Click **Play** on any game to open the overlay.
- Use **Visit Original Site** if the game cannot embed.
- Search games with the search bar by typing keywords.
- Adjust background music volume at the bottom-right.
